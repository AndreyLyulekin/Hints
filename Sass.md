Препроцессоры CSS, такие как Sass, Less или Stylus, позволяют разработчикам писать CSS код более эффективно и удобно.
Постпроцессоры CSS, например Autoprefixer, CSSNano или PostCSS,
используются для улучшения CSS кода после его написания.
Могут:
● добавлять вендорные префиксы для кросс-браузерной совместимости
● минифицировать код

CSS модули - это методология структурирования CSS кода
которая позволяет изолировать стили для разных компонентов
через автоматическую генерацию уникальных имен классов

● Переменные: Sass позволяет использовать переменные для хранения значений
$primary-color: #ff0000;

body {
background-color: $primary-color;
}

● Вложенность: Sass позволяет вкладывать правила CSS, что делает структуру стилей более читаемой и логичной.
nav {
ul {
list-style: none;
}

li {
display: inline-block;

    a {
      text-decoration: none;

      &:hover {
        color: #ff0000;
      }
    }

}
}

● Миксины: полуфункции (декомпозированная логика) для повторного использования.
@mixin border-radius($radius) {
border-radius: $radius;
}

button {
@include border-radius(5px);
}

● Встроенные функции: функции для работы со строками, цветами, математическими операциями

$base-font-size: 16px;

.container {
font-size: $base-font-size + 2px;
color: darken(#ff0000, 20%);
}

1. Функции для работы со строками:

- str-length($string) - возвращает длину строки.
- str-insert($string, $insert, $index) - вставляет подстроку в указанное место строки.
- str-index($string, $substring) - возвращает индекс первого вхождения подстроки в строку.

2. Функции для работы с цветами:

- lighten($color, $amount) - увеличивает яркость цвета на указанное количество процентов.
- darken($color, $amount) - уменьшает яркость цвета на указанное количество процентов.
- rgba($color, $alpha) - задает прозрачность для цвета, возвращая новый цвет RGBA.

3. Функции для работы с числами:

- round($number) - округляет число до ближайшего целого.
- abs($number) - возвращает абсолютное значение числа.
- percentage($number) - преобразует число в процентное значение.

4. Функции для работы со списками:

- length($list) - возвращает длину списка.
- nth($list, $index) - возвращает элемент списка по указанному индексу.
- join($list1, $list2, $separator) - объединяет два списка с указанным разделителем.

5. Другие полезные функции:

- if($condition, $if-true, $if-false) - выполняет условное выражение и возвращает одно из двух значений, в зависимости от условия.
- random() - генерирует случайное число в диапазоне от 0 до 1.
